# bare-cua

Rust fuzzing and testing workspace for CUA (Cognitive User Assistant) patterns.

## Quick Start

```bash
cargo test --all-features --workspace
cargo clippy --all-features -- -D warnings
```

## Dependencies

- Rust stable
- cargo-fuzz (for fuzzing targets)
